		*****REACT TO DO APP*****

--A simple to do list made with REACT--

Instructions:

-Download ZIP file and unzip in the same folder.
-Open in a text editor of your choice
-Run a live server
-Enjoy!


!!!If your text editor doesn't have the live server functionality run the folowing steps:

-Download Nodejs from https://nodejs.org/en/
-Open any command prompt console
-run npm install -g http-server //(can be sudo npm install -g http-server)
-run http-server -o
-Enjoy!
